## Get Started in 3 Steps

### Step 1: Install (1 minute)
```bash
npm install
```

### Step 2: Run (1 minute)
```bash
npm run dev
```

### Step 3: Visit (seconds)
Open your browser to: http://localhost:3000

Done! You're now running the portal.

## Want to Customize?

### Change Colors
Edit `tailwind.config.ts` and update color hex values.

### Add Superapp
Edit `src/app/page.tsx`, add to the `superapps` array.

### Adjust Animations
Edit animation props in `src/app/page.tsx`.

See **README.md** for full customization guide.
